# VYNO Process Guard: Industrial AI Monitoring Platform

VYNO Process Guard is an enterprise-grade B2B SaaS platform designed to monitor factory telemetry in real-time. Using Predictive Reasoning (Llama-3.1), the system detects mechanical anomalies and provides prescriptive engineering actions.

## 🚀 Core Features

- **Multi-Tenant Architecture:** Cryptographically isolated data pipelines for different organizations.
- **Real-time Telemetry:** Sub-second data visualization using WebSockets (Supabase Realtime).
- **Prescriptive AI:** Actionable insights generated by Llama-3.1 via Groq API.
- **EU AI Act Ready:** Every AI decision is hashed and logged in an immutable audit trail.

## 🛠 Tech Stack

- **Frontend:** Next.js 14 (App Router), Tailwind CSS, Recharts.
- **Backend:** FastAPI (Python), Groq SDK (Llama-3.1-8b-instant).
- **Database/Auth:** Supabase (PostgreSQL with RLS).

## 🛡 Security & Compliance

The platform utilizes **Row Level Security (RLS)** to ensure data isolation. Every telemetry packet is stamped with a unique `user_id` and verified at the database layer.
